&^ 运算符

November 16, 2022 · 1 min · 冲和
Table of Contents
  • 含义
  • 示例

在查看 Mutex 源码中，看到了 &^ 这个运算符， 没见过，就去查了查。

含义#

按位清零

即：

  • 运算符右边数字二进制为1的位置，左边数字二进制对应位置置0
  • 运算符右边数字二进制为0的位置，左边数字二进制对应位置保留

示例#

a := 10
b := 2
res := a &^ b

fmt.Printf("a = %b(%d)\n", a, a) // a = 1010(10)
fmt.Printf("b = %b(%d)\n", b, b) // b = 10(2)
fmt.Printf("res = %b(%d)\n", res, res) // res = 1000(8)
