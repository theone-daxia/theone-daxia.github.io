Hugo:静态博客网站搭建记录

目标

使用 GithubPages、Hugo 创建个人博客网站

功能

  • 正常博客写作
  • 文章支持评论
  • 网站数据分析

准备工作

  • 拥有一个 github 账号
  • 安装 hugo（ 参考文档：https://gohugo.io/getting-started/installing/ ）

步骤

github 创建两个仓库

  • 一个源码仓库，设置权限为 Private（用于对网站进行任何更改操作）
  • 一个博客仓库，设置权限为 Public（用于对外展示，项目名称为：xxx.github.io，xxx为账户名称）

初始化源码项目

  1. 使用 hugo 生成网站
hugo new site quickstart
  1. 选择并添加一款心仪主题（ 主题选择：https://themes.gohugo.io/ ）
# 此处以 Base16 主题为例

cd quickstart
git submodule add
  1. 添加博客内容
hugo new blogs/my-first-blog.md

  1. 配置菜单

  2. 将 quickstart 目录里的内容 push 到源码项目，完成项目初始化

配置 CI/CD

可采用 hugo -D 来生成静态页面，然后同步到公开的博客仓库。 但这明显不符合极客精神，所以采用 GithubActions 来自动集成部署。
